# 🖼️ VisionCaption AI – Image Captioning

VisionCaption AI is an Artificial Intelligence web application that generates meaningful captions for uploaded images using a pre-trained deep learning model. The project is built with **Python, Flask, HTML, CSS, JavaScript, and Hugging Face Transformers**.

## 🚀 Installation

1. Clone or download the project.

2. Open the project folder.

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000
```

---

## 🧠 AI Model

This project uses the **Salesforce BLIP Image Captioning Model** from Hugging Face. It analyzes uploaded images and generates descriptive captions using a pre-trained transformer-based deep learning model.

---

## 📷 How to Use

1. Open the application.
2. Upload an image.
3. Click **Generate Caption**.
4. Wait a few seconds.
5. View the AI-generated caption.
